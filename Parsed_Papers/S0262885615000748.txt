@&#INTRODUCTION@&#

Web images, most of which are unlabeled, have shown continuous explosive growth. As an important means, semi-supervised sparse feature selection [1–4] has the ability to improve the performance of web image annotation. It has extensively shown that semi-supervised sparse feature selection approaches can overcome the drawbacks of supervised feature selection methods and unsupervised feature selection methods. On the one hand, semi-supervised sparse feature selection approaches can save human labor cost for labeling a large amount of training data, and on the other hand, they can make full use the reliable labeled data and the accessible unlabeled data simultaneously to improve the sparse feature selection performance.

Among different semi-supervised learning methods, graph Laplacian regularization based method is one of the most representative works [5]. The graph Laplacian can determine the geometry of the underlying manifold in Laplacian regularization. Now, the graph Laplacian regularization based semi-supervised learning has been widely applied into semi-supervised sparse feature selection [1,2]. In [1], Ma et al. have proposed a structural feature selection with sparsity frame based on graph Laplacian semi-supervised learning to select features with considering the correlation between them. In [2], Shi et al. have proposed a semi-supervised sparse feature selection method based on graph Laplacian and l
                     2,1/2-matrix norm to select more sparse and discriminative features for image annotation. In this paper, we also exploit graph Laplacian regularization to construct our semi-supervised sparse feature selection frame.

As we know, images are usually represented by different types of features, such as color correlogram, wavelet texture, and edge direction histogram. Each type of features characterizes these images in one specific feature space and has particular physical meaning and statistic property. Conventionally, the data represented by multiple types of features are named as multi-view data to distinguish from the single-view data represented only by one type of features [6]. However, most of the existing semi-supervised sparse feature selection methods are developed for the single-view data and these methods concatenate multiple views features into a long vector once they confront with multi-view data, such as [1,2]. This concatenation strategy cannot efficiently explore the complementary of different view features because it improperly treats different view features carrying different physical characteristics. In addition, this concatenation strategy ignores the physical interpretations of different features and results in an over-fitting problem when the size of the training set is small.

Recent research has shown that multi-view learning can address the above problems and leverage the correlated and complementary information between different views, thus multi-view learning has obtained more and more research interest and it has been explored in variety of applications. Xu et al. have finished a survey on multi-view learning in their paper [7]. In [8], Liu et al. have applied multi-view learning in the sparse coding for image annotation. In [9], Liu et al. have proposed multi-view regularization to support vector machines for image annotation. In [10], Xu et al. have proposed large-margin multi-view Gaussian process for image classification. In [11], Luo et al. have proposed vector-valued multi-view semi-supervised learning for multi-label image classification. In [12], Xia et al. have proposed multi-view spectral embedding for image retrieval, video annotation, and document clustering. In [13], He et al. have introduced multi-task multi-view learning for complicated learning problems with both feature heterogeneity and task heterogeneity. In [6], Feng et al. have proposed an adaptive unsupervised multi-view feature selection for visual concept recognition. However, as far as I know, multi-view learning has not been widely used in semi-supervised sparse feature selection. Therefore, we apply multi-view learning into our method to boost the semi-supervised sparse feature selection performance.

In this paper, we present a semi-supervised sparse feature selection method based on multi-view Laplacian regularization, namely, multi-view Laplacian sparse feature selection (MLSFS). The framework of MLSFS for web image annotation is illustrated in Fig. A
                     . Multi-view learning, which can effectively explore the complementation of different features from different views, is introduced to MLSFS for forming the multi-view Laplacian regularization to improve the performance of semi-supervised sparse feature selection. An effective iterative algorithm is proposed to optimize the objective function. We apply the proposed MLSFS into web image annotation and some experiments are conducted on two large-scale web image datasets, namely, NUS-WIDE dataset [14] and MSRA-MM2.0 dataset [15]. The results demonstrate that MLSFS outperforms state-of-the-art single-view semi-supervised sparse feature selection methods and unsupervised feature selection methods.

The main contributions of our work are as follows:
                        
                           •
                           Multi-view Laplacian sparse feature selection (MLSFS) method is proposed, which exploits the multi-view learning to build multi-view Laplacian regularization for boosting the semi-supervised sparse feature selection performance.

An effective iterative algorithm is proposed to optimize the objective function and its convergence is analyzed.

Some experiments are conducted and the results indicate that MLSFS have better performance than single-view semi-supervised sparse feature selection methods and unsupervised feature selection methods.

The rest of this paper is organized as follows. Section 2 reviews the multi-view learning and semi-supervised learning. Section 3 elaborates MLSFS framework and its convergence. Section 4 conducts extensive experiments on two datasets to evaluate the performance of MLSFS. Lastly, Section 5 concludes this paper.

@&#RELATED WORK@&#

In this section, we review the related work of our MLSFS algorithm, including multi-view learning and semi-supervised learning.

Recently, multi-view learning has obtained extensive research interest and the existing multi-view learning algorithms can be grouped into the following four categories: co-training, multiple kernel learning, subspace learning, and graph-based learning.

Co-training [16] is one of the earliest schemes for multi-view learning. It trains alternately to maximize the mutual agreement on two distinct different views of the unlabeled data. Co-training method effectively exploits unlabeled data to improve the performance, especially when the two views are conditionally independent of one another [17]. In recent years, co-training has been extended to EM and SVM to form co-EM [18] and co-SVM [19].

Multiple kernel learning (MKL) algorithms learn a kernel machine from multiple Gram kernel matrices [20–22], which naturally correspond to different views of features and are combined either linearly or non-linearly to improve learning performance. In [20], Lanckriet et al. have formulated MKL as a semi-definite programming problem. In [21], Francis et al. have treated MKL as a second order cone program problem and developed a sequential minimal optimization (SMO) algorithm to efficiently obtain the optimal solution. In [22], Sonnenburg et al. have reformulated the binary classification MKL problem as a semi-infinite linear programming problem and made MKL applicable to large scale data analytics problems.

Subspace learning-based algorithms aim to obtain a latent subspace shared by multiple views by assuming that the input views are generated from this latent subspace. Given this subspace, it is straightforward to conduct the subsequent tasks, such as classification and clustering. Canonical correlation analysis (CCA) [23] and kernel canonical correlation analysis (KCCA) [24] are the representative methods, which explore basis vectors for two sets of variables by mutually maximizing the correlations between the projections onto these basis vectors. These methods can be straightforward applied into two-view data to select the shared latent subspace, and then they have been further developed to conduct multi-view clustering [25] and regression [26].

Graph-based algorithms firstly utilize a special graph to encode the local geometry of a particular view, and then integrate all the graphs for different views to explore the complementary properties of them. Some typical models include [12, 27–29]. In [12], Xia et al. have proposed multi-view spectral embedding for data clustering, where a set of optimal coefficients have been used to linearly combine different graphs corresponding to different views. In [28], Xie et al. have proposed a multi-view stochastic neighbor embedding for data visualization, which integrate multiple graphs corresponding to different views based on the robust stochastic gradient descent method.

Facing the large number of unlabeled data and small number of labeled data, semi-supervised learning algorithms have been widely researched and many algorithms have been proposed, which can be grouped into three categories: generative models, transductive support vector machines (TSVM), and graph-based methods.

Generative models are perhaps the oldest semi-supervised learning method and they assume that examples are clustered and generated by the same parametric model. These models employ expectation–maximization algorithm [30] to learn the missing model parameters. MG [31], ME [32], and NB [33] are the representative methods.

TSVM [34] is the typical semi-supervised method, which utilizes the labeled data to initiate SVM and then assigns the potential labels to unlabeled data. However, TSVM has two main drawbacks: its loss function is non-convexity and its computation complexity is high. So some improved methods have been proposed, such as [35,36].

Graph-based models define a graph where the nodes are labeled and unlabeled training examples and the edges reflect the similarity of examples. These models assume label smoothness over the graph and which are nonparametric, discriminative, and transductive in nature [5]. According to the regularization framework [37], a lot of graph-based methods can be obtained by defining different loss functions and regularizations, among which the graph Laplacian based methods are the typical model [38–40].

@&#EXPERIMENTS@&#

In this section, extensive experiments are conducted to validate the performance of the proposed MLSFS algorithm by applying it into web image annotation task.

Two web image datasets, i.e. NUS-WIDE dataset [14] and MSRA-MM2.0 dataset [15] are used in our experiments. The NUS-WIDE dataset was created by the Lab for Media Search in National University of Singapore in 2009, including 269648 real-world images with 81 concepts. The MSRA-MM2.0 dataset was created by Microsoft Research Asia also in 2009, consisting of 50000 images with 100 concepts. In our experiments, we use three types of visual features, i.e. color correlogram (CORR), edge direction histogram (EDH), and wavelet texture (WT). Table A
                      shows the dimensions of these features on these two datasets.

In our experiments, we randomly sample 3000 images as the training set and use the rest of the images as the testing set for each dataset. In the training set, the labeled samples account for 10%, 20%, 30%, 50%, 70%, and 90%, respectively. We observe the performance variation according to the number of labeled training data and record the corresponding results. The experiments are conducted ten times independently and the average results are reported. To evaluate the annotation performance, we use two metrics, i.e. mean average precision (MAP) and recall. Following [43,44], recall can be calculated with a fixed annotation length of 10 labels per image.

There are several parameters needed to tune in our experiments. The number of selected features ds, which is set to 100, 150, 200, 250, 300, and all respectively. The number of nearest neighbors to compute the Laplacian matrix k is set to 25 in all experiments. Two regularization parameters μ and λ in Eq. (7) are tuned by a “grid-search” strategy from the range of {0.00001, 0.001, 0.1, 1, 10, 1000, 100000}. Parameter γ is set to 1010 empirically as it is not sensitive.

We compare our proposed MLSFS method with some related sparse feature selection methods to evaluate its performance. There are four single-view semi-supervised sparse feature selection methods, including SBMLR [45] based on Bayesian l
                     1 regularization, FSNM [46] based on joint l
                     2,1-norms minimization, SFSS [1] based on graph laplacian and l
                     2,1-norm, and FSLG [2] based on graph Laplacian and l
                     2,1/2-matrix norm, which concatenate different types of features into a long vector when they confront multi-view data. In addition, we compare MLSFS with another multi-view unsupervised sparse feature selection method AUMFS [6], namely, adaptive unsupervised multi-view feature selection.

@&#PERFORMANCE EVALUATION@&#

We compare MLSFS with related semi-supervised sparse feature selection methods when the labeled data are at different percentages, including 10%, 20%, 50%, and 90%, respectively. Ten times independent experiments are conducted and the average results are reported. Table B
                         shows the compared results according to MAP and Table C
                         shows the compared results according to recall. The best results are in bold.


                        Tables B and C show us the following observations: 1) No matter what the percentage of labeled images is, the proposed MLSFS method outperforms all the compared methods, which concatenate different types of features into a long vector without considering the correlated and complementary information between different views. 2) As the number of labeled training images increases, the performance of all the methods gradually improves. From Table B, we can see that when the percentage increases from 10% to 20%, the MAP of MLSFS increases from 0.071 to 0.087 on NUS-WIDE dataset while the MAP increases from 0.053 to 0.058 on MSRA-MM dataset. At the same time, we can see from Table C that when the percentage increases from 10% to 20%, the recall of MLSFS increases from 0.596 to 0.657 on NUS-WIDE dataset while the MAP increases from 0.637 to 0.655 on MSRA-MM dataset.

The observations indicate that the proposed MLSFS can effectively select the most discriminative sparse features to boost the annotation performance. The main attribution is that MLSFS exploits the multi-view learning, which effectively explores the complementation of different features from different views. In addition, MLSFS adopts the graph Laplacian based semi-supervised learning to simultaneously utilize the reliable labeled data and the accessible unlabeled data to improve sparse feature selection performance. Lastly, the proposed MLSFS is built upon l
                        2,1/2-matrix norm, which makes our model select the most discriminative sparse features with good robustness.

In order to evaluate the performance of multi-view learning, we compare the proposed MLSFS with single-view feature selection method FSLG with the concatenated-view features and different view features respectively in this experiment. The compared results are shown in Fig. B
                        , where CORRL, EDHL and WTL denote the performance of FSLG by using the single view features CORR, EDH and WT respectively. This experiment is conducted on two datasets with labeled training data at different percentage, including 10%, 20%, 30%, 50%, 70%, and 90%. MAP is used as the metric to evaluate the performance.

From Fig. B, we can see that MLSFS have the best performance on two datasets, which shows that the MLSFS method with multi-view features outperforms the method FLSG with single-view features and concatenated features. This indicates that the multi-view learning exploited by MLSFS can effectively explore the complementation of different features from different views, and then to boost the performance of the sparse feature selection.

In this section, we conduct an experiment to evaluate the performance of sparse feature selection with different numbers of the selected features. At the same time, we compare MLSFS with AUMFS in this experiment. The number of the selected features is set to 100, 150, 200, 250, 300, and all, respectively. The parameter μ is set to 0.00001 and λ is set to 1 on NUS-WIDE dataset, while the parameters μ and λ are all set to 10 on MSRA dataset. Here we use 10% percentage labeled training data and MAP is used as the metric. The results are shown in Fig. C
                        .


                        Fig. C illustrates the performance of MLSFS and AUMFS varies when the number of selected features changes. From Fig. C, we have the following observations. 1) At first, the MAP of MLSFS and AUMFS both increase with the number of selected features increasing, and then they decrease with the number of selected features increasing after arriving the peak. This could be attributed to the loss of some useful information with too sparse selected features and the redundant features and noise with all the features. 2) The MAP of MLSFS achieves the peak with 250 selected features while the MAP of AUMFS achieves the peak with 200 selected features on NUS-WIDE dataset. The MAP of MLSFS achieves the peak with 200 selected features while the MAP of AUMFS achieves the peak with 250 selected features on MSRA dataset. 3) The performance of MLSFS based on semi-supervised learning is far better than the performance of AUMFS based on unsupervised learning no matter how many features selected on both NUS-WIDE dataset and MSRA dataset. This because MLSFS can make full use the reliable labeled data and the accessible unlabeled data simultaneously to improve the sparse feature selection performance. From the observations, we conclude that MLSFS can select the most sparse and discriminative features and reduce noise to achieve the better annotation performance.

The regularization parameters μ and λ in Eq. (7) can influence the performance of MLSFS. Here we conduct an experiment to study the influence of different choices of them. Parameters μ and λ are tuned by a “grid-search” strategy from the range of {0.00001, 0.001, 0.1, 1, 10, 1000, 100000}. Following the above experiment, 10% percentage labeled training data are used and MAP is used as the metric. We conduct this experiment on both NUS-WIDE dataset and MSRA-MM2.0 dataset and the experimental results are shown in Fig. D
                        .


                        Fig. D demonstrates that our model MLSFS is sensitive to these two regularization parameters μ and λ on the two datasets and different choices of μ and λ result in different annotation performance. When μ
                        =0.00001 and λ
                        =1, MLSFS can achieve the best annotation performance on NUS-WIDE dataset, while when μ and λ are approximatively equal, MLSFS can achieve the best annotation performance on MSRA dataset, especially we set μ
                        =
                        λ
                        =10 in our other experiments.

In this section, we experimentally prove the convergence of MLSFS by conducting an experiment on NUS-WIDE dataset and MSRA, respectively. Here 10% percentage labeled training data are used and the number of selected features is 250 and 200, respectively, on NUS-WIDE dataset and MSRA. This experiment is conducted on a personal computer (2.90 GHz, 4Gb RAM) with Matlab (2010b) and the convergence curves are shown in Fig. E
                        .

This experiment verifies the convergence of MLSFS and the convergence curves are shown in Fig. E. It can be observed from Fig. E.1 that the objective function value converges via 5 iterations on NUS-WIDE dataset, meanwhile it can be observed from Fig. E.2 that the objective function value converges via 4 iterations on MSRA dataset. In addition, the convergence time is 26.4 seconds and 23.9 seconds respectively. The results illustrate the proposed MLSFS method has good convergence, which indicates MLSFS can select more sparse and discriminative features and is capable to annotate large-scale web images.

@&#CONCLUSION@&#

In this paper, we introduce the multi-view learning into semi-supervised sparse feature selection and propose MLSFS frame based on multi-view Laplacian regularization. MLSFS can realize the sparse feature selection well by exploiting the correlated and complementary information of different features from different views. In addition, MLSFS adopts the graph Laplacian based semi-supervised learning to simultaneously utilize the reliable labeled data and the accessible unlabeled data to improve sparse feature selection performance. We design an efficient iterative algorithm to optimize the objective function and prove its convergence theoretically and experimentally. We apply MLSFS into web image annotation and several experiments are performed on two image datasets. The experimental results demonstrate that MLSFS algorithm has the ability to boost the sparse feature selection performance.
